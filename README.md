# Rishit Luthra — Interactive Portfolio

A gamified, recruiter-friendly portfolio built with Next.js, Framer Motion, and GSAP.

## Features

- **Portfolio Progress** — Section exploration tracker with 100% completion reward (resume download)
- **XP System** — Earn points while exploring; floating level badge
- **Recruiter Quest** — Optional challenge mode with achievement badge
- **Skill Tree** — Interactive gaming-style skill visualization
- **Project Arena** — Missions with difficulty, category, and completion animations
- **Career Journey Map** — GSAP-animated milestone path
- **Trophy Cabinet** — Glassmorphism badge collection
- **AI Assistant NPC** — Guided exploration chatbot
- **Easter Eggs** — Konami code, particle clicks, avatar double-click
- **Premium Motion** — Particles, confetti, card tilt, mouse gradient, sound toggle

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- Update contact links in `src/app/page.tsx`
- Edit content in `src/data/portfolio.ts`
- Replace `public/avatar.png` and `public/resume.pdf`

## Build

```bash
npm run build
npm start
```
